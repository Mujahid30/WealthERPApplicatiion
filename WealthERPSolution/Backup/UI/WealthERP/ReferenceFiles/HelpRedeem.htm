Redeem
The existing AMC and the existing scheme will be displayed. Select the same in order to redeem (Sell) your units.
AMC: Asset Management Company. This will reflect the list of AMC’s you hold. Select the particular AMC in which you wish to place your redemption (sell) order.
Category : This will reflect the various categories available for the respective AMC’s.
Scheme: This will reflect the list of schemes available of the respective AMC’s available for redemption.
Last recorded NAV: Net Asset Value recorded as on previous day.
Cut- off time: It shows the NAV that will be applied to you. If you place the order post the cut off time, NAV applicable will be of the same day. If the order is placed before the cut-off time, NAV is of the previous day. Also, if the order is placed before the cut-off time, the order will be processed on the same day, if it is placed after the cut-off time, the order will be processed the next trading day.
Subsequent Amount (In multiples of): This indicates the smallest permissible additional purchase, a fund will accept.
Scheme Rating: This will display the rating for the fund by Morning Star – a leading independent investment research company globally.
Folio Number:For the additional purchase, select the existing folio number.
Amount: Fill in the total amount you wish to invest. You will find a drop down with various options: Amount, Units and All Units can be selected. If you select – Amount, fill in the amount you wish to redeem, if you select units, fill in the number of units you wish to redeem, if you select All units, the amount column need not be filled and all units can be redeemed (sold).
